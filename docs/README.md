# wedesign
## Depedencies
```
# Nodejs using Ubuntu
curl -fsSL https://deb.nodesource.com/setup_current.x | sudo -E bash -
sudo apt-get install -y nodejs

# Vuejs 
npm install -g @vue/cli

```

```
npm install
```
